# Reactflix

A simple frontend clone of a movie streaming service, built with React and powered by the TMDB API. This project demonstrates how to fetch data from an external API and display it in a responsive grid layout.

## Features
* **Browse Movies:** View lists of trending, top-rated, and popular movies.
* **Responsive UI:** Styled with pure CSS to mimic the "Netflix" look and feel.
* **Live Data:** Real-time movie data fetching using The Movie Database (TMDB) API.

## Tech Stack
* **Frontend:** React.js (JavaScript)
* **Styling:** CSS3
* **Data Source:** [TMDB API](https://www.themoviedb.org/documentation/api)

## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/arshcwb/reactflix.git
   cd reactflix/frontend

2. **Install dependencies**

   ```bash
   npm install


3. **Start the web-app**
    ```
   npm start
Open the given local port address to view it in the browser.

4. **Configure API Key**

- Get a free API Key from TMDB.
- Open the web-app, it will ask you to submit a valid TMDB API.
- Add your key in the form input.
